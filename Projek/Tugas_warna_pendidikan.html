Pemrograman Web Dasar dan Bootstrap

Pemrograman Web Dasar adalah fondasi yang wajib dikuasai untuk membangun segala jenis situs web, terdiri dari:

  1. HTML (HyperText Markup Language): Bertanggung jawab atas struktur dan konten halaman (kerangka).
  2. CSS (Cascading Style Sheets): Bertanggung jawab atas gaya dan tampilan visual halaman.
  3. JavaScript (JS): Bertanggung jawab atas interaktivitas dan fungsionalitas dinamis.

Bootstrap adalah Framework CSS yang membantu Anda membangun website responsif dengan cepat tanpa perlu membuat semua CSS dari nol.

Pelajari Lebih Lanjut

Desain Web: Estetika dan Pengalaman Pengguna (UX/UI)

Desain Web berfokus pada tampilan dan pengalaman pengguna dari sebuah situs web. Desain yang baik memastikan situs Anda tidak hanya indah (UI - User Interface) tetapi juga mudah digunakan dan dinavigasi (UX - User Experience).

Dalam konteks Bootstrap, desain web mencakup penggunaan Grid System, pemilihan tipografi, warna, dan tata letak yang konsisten dan adaptif.

Elemen Kunci Desain Responsif:

  • Responsif: Menggunakan sistem Grid 12 kolom untuk adaptasi ke semua perangkat.
  • Konsistensi: Memanfaatkan komponen (Navbar, Card, Button) yang seragam.
  • Aksesibilitas: Memastikan kontras warna dan ukuran teks memadai.

Database Management System (DBMS)

DBMS (seperti MySQL, PostgreSQL, atau MongoDB) adalah perangkat lunak yang memungkinkan Anda untuk berinteraksi dengan database. Ini adalah pilar krusial untuk situs web dinamis (seperti e-commerce atau blog) yang harus menyimpan dan mengelola data.

Tanpa DBMS, setiap data yang dimasukkan oleh pengguna akan hilang saat halaman dimuat ulang. DBMS menyediakan sistem terstruktur untuk memastikan data aman dan konsisten.

Pentingnya DBMS dalam Web Development:

  • Penyimpanan Data: Menyimpan data secara persisten dan aman.
  • Query Data: Memungkinkan pengambilan dan manipulasi data spesifik menggunakan SQL.
  • Integritas: Memastikan hubungan antar data (relasi) tetap valid.
